# 前言
本网站的内容主要来自于[Apache® Ignite™社区](https://ignite.apache.org)的[文档库](https://apacheignite.readme.io/docs)，非常感谢社区给予的支持！

Apache Ignite是一个可以水平扩展的、容错的分布式内存计算平台，可以用于创建以内存级速度处理TB级数据的实时应用。
